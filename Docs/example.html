CSS: Способы подключения стилей и селекторы

Способы подключения стилей

1. Внешний файл (рекомендуемый способ)

<link rel="stylesheet" href="style.css">

Подключение отдельного CSS-файла в разделе <head> документа.

2. Внутренние стили

<style> div { color: red; } </style>

Определение стилей внутри тега <style> в разделе <head>.

3. Встроенные стили (инлайновые)

<div style="color: red;"></div>

Определение стилей непосредственно в атрибуте style HTML-элемента.

Пример специфичности селекторов

Селектор Специфичность Пример
Элемент (p, div) 0,0,1 p { color: blue; }
Класс (.class) 0,1,0 .text { color: red; }
ID (#id) 1,0,0 #header { color: green; }
Инлайновые стили 1,0,0,0 <p style="color: purple;">
!important Наивысший p { color: orange !important; }

Селекторы

Демонстрация селекторов

<p id="par"> Параграф с <b>ipsum</b> dolor. </p> <span class="cls"> Sit amet <b>consectetur</b>. </span>

По тегу:

p { color: red; }

Параграф с ipsum dolor.

Sit amet consectetur.

По классу:

.cls { background-color: #3498db; color: white; padding: 5px; }

Параграф с ipsum dolor.

Sit amet consectetur.

По ID:

#par { background-color: #2ecc71; color: white; padding: 5px; font-weight: bold; }

Параграф с ipsum dolor.

Sit amet consectetur.

Комбинированные селекторы:

/* Все элементы b внутри параграфов */ p b { text-decoration: underline; } /* Все элементы b внутри элементов с классом cls */ .cls b { font-style: italic; }

Параграф с ipsum dolor.

Sit amet consectetur.

Популярные селекторы

Селектор Пример Описание
* * { margin: 0; } Выбирает все элементы
элемент p { color: blue; } Выбирает все элементы <p>
.class .intro { font-weight: bold; } Выбирает все элементы с class="intro"
#id #header { height: 100px; } Выбирает элемент с id="header"
элемент, элемент div, p { margin: 10px; } Выбирает все <div> и <p> элементы
элемент элемент div p { color: red; } Выбирает все <p> внутри <div>
элемент > элемент div > p { color: blue; } Выбирает все <p>, где родитель <div>
элемент + элемент div + p { margin-top: 0; } Выбирает <p>, который следует сразу после <div>
